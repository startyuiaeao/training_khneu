# Перукарня CRM (training_khneu)

Даний проект розроблений в рамках навчального тренінгу. Це веб-додаток на базі Django для автоматизації роботи мережі перукарень. Система управляє базою клієнтів, враховує знижки лояльності та веде каталог послуг. Також реалізована підтримка філій, гнучке ціноутворення та облік виконаних робіт з генерацією чеків.

## 🌟 Основний функціонал

- **Управління клієнтами:** Ведення бази клієнтів, облік кількості візитів, автоматичне присвоєння категорій (наприклад, "Постійний") та знижок лояльності.
- **Каталог послуг (стрижок):** Створення та редагування видів стрижок з урахуванням статі, тривалості та актуального статусу.
- **Мережа філій:** Додавання філій перукарні з адресами та контактними даними.
- **Гнучке ціноутворення:** Можливість створювати окремі прайс-листи для різних філій з прив'язкою до дат дії (історія зміни цін).
- **Журнал обліку послуг:** Фіксація виконаних робіт, автоматичний розрахунок кінцевої вартості з урахуванням знижки клієнта та генерація номерів чеків.
- **Рахунки-фактури:** Можливість перегляду та друку деталізованих чеків (рахунків-фактур) для клієнтів.
- **Звіти та аналітика:**
  - Загальний звіт про надані послуги (доходи по всій мережі).
  - Деталізований звіт по обраних філіях за визначений період.

## 🛠 Технологічний стек

- **Backend:** Python 3, Django
- **База даних:** SQLite (за замовчуванням)
- **Frontend:** HTML, CSS, JavaScript, Bootstrap 5, Bootstrap Icons
- **Інше:** Спеціальні Management-команди для заповнення бази даних.
